### Demo Scenario: Turbocharger Housing Optimization

#### Initial Setup (5 minutes)
1. Connect Clawdbot to ProCAST installation
2. Upload baseline turbocharger CAD model
3. Define optimization objectives (minimize porosity, reduce weight)
4. Set material parameters (stainless steel grades)

#### Live Demonstration (15 minutes)
1. **Baseline Simulation**
   - Clawdbot sets up and runs initial simulation
   - Identifies critical areas: hot spots, potential porosity
   - Generates baseline performance report

2. **Automated Optimization Loop**
   - AI suggests gating system modifications
   - Runs 5 design variants in parallel
   - Compares results in real-time dashboard
   - Recommends optimal design

3. **Knowledge Integration**
   - Queries material database for alternative alloys
   - Searches literature for similar case studies
   - Applies best practices for thin-wall castings

4. **Report Generation**
   - Executive summary with key findings
   - Technical details with simulation imagery
   - Cost-benefit analysis
   - Next steps recommendations
